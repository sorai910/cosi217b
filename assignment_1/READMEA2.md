# Requirements
Python 3.10.9 is needed to run this program.
Use the following command to install required dependecies
```bash
$ pip install {dependency}
```
- dependeicies:
  - sqlite
  - flask
  - sqlalchemy
  - spacy
  - flask_sqlalchemy
# How to run
- alchemy_app.py
```bash
$ python3 alchemy_app.py
```
